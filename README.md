# React Spring Boot Login System

Full Stack Authentication System using:

- React.js
- Spring Boot
- MongoDB Atlas
- Email OTP Verification

## Features
- User Signup
- User Login
- Forgot Password
- Email OTP Verification
- Reset Password

## Tech Stack
Frontend: React  
Backend: Spring Boot  
Database: MongoDB Atlas  
Email Service: JavaMailSender

## Run Project

Backend:
mvn spring-boot:run

Frontend:
npm install
npm start
